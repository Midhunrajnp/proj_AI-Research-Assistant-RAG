# proj_AI-Research-Assistant-RAG# AI Research Assistant using RAG

## Overview

An AI-powered Research Assistant that uses Retrieval-Augmented Generation (RAG) to answer questions from PDFs, research papers, web pages, and YouTube transcripts.

## Features

- Gemini 2.5 Flash Integration
- Gemini Embeddings
- ChromaDB Vector Database
- BM25 Retrieval
- Hybrid Search
- Reciprocal Rank Fusion
- Citation-backed Responses

## Tech Stack

- Python
- Gemini API
- ChromaDB
- LangChain
- BM25
- Google Colab
